## Using PipEnv Setup

**i). Install PipEnv**

Check version to make sure you are running python3

``python --version``

Proceed to install PipEnv

``pip3 install pipenv``

**ii) Activate Env Shell**

``pipenv shell``

**iii) Install Packages**

``pipenv install``
